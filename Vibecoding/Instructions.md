I want to create an app (for linux, mac and windows) that has the same capabilities as Insync (https://www.insynchq.com/).

The app should be able to sync files to andfrom Google Drive. to a local directory.

The user configures the local folder and the Google Drive folder to sync.
Multiple syncs can be configured.

App should be fast and lightweight.
App should give the user good feedback on issues and progress.

Use Rust for the backend and frontend.
